## Описание

Консольное приложение для комплексного управления личными финансами:
- Учёт доходов и расходов.
- Отдельный баланс накоплений с возможностью перевода между счетами.
- Планирование и автоматическое выполнение регулярных платежей.
- Детальная статистика по месяцам с визуализацией долей.

Проект полностью переписан по сравнению с первой версией: теперь это объектно-ориентированная система с разделением ответственности и независимыми хранилищами данных.

## Ключевые возможности

### 1. Управление транзакциями
- Добавление дохода или расхода с автоматической фиксацией даты.
- Просмотр всех записей или фильтрация по диапазону дат.
- Удаление транзакции по уникальному идентификатору.

### 2. Сбережения
- Собственный баланс, независимый от основного счёта.
- Прямое пополнение накоплений.
- Перевод средств с основного баланса в накопления и обратно.
- История операций (полная или за период).

### 3. Регулярные платежи
- Создание повторяющегося дохода или расхода с указанием дня месяца, суммы и описания.
- Настройка автоматического подтверждения (да/нет).
- При запуске программы система проверяет, не был ли платёж уже проведён сегодня (`last_processed`), и при необходимости выполняет его (автоматически или запрашивая разрешение).

### 4. Статистика
- Расчёт доходов и расходов за любой выбранный месяц (формат `ГГГГ-ММ`).
- Детализация расходов с расчётом доли каждой категории в процентах.
- Визуальная шкала: каждая группа символов `|` соответствует 5% от общей суммы расходов.

### 5. Текущий баланс
- Отображает остаток основного счёта, учитывая все доходы, расходы и переводы между счетами (сбережения не включены в основной баланс).

### 6. Автосохранение
- Каждое изменение немедленно записывается в соответствующий JSON-файл. Ручное сохранение не требуется.
